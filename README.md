# Beem_Website_POC
This project automates UI test scenarios for an Fintech website using Selenium and TestNG. 
It demonstrates an end-to-end automation framework covering login, Subscription, Subscription management and home page.

## Project structure
| Folder/File| Description|
|------------|-------------------------|
| 'src/main/java/base' | Contains base classes (Driver setup, Config reader ect.) |
| `src/main/java/pages` | Page Object classes for each web page |
| `src/main/java/utilities` | Reusable utilities (Excel reader, Waits, API helpers) |
| `src/test/java/tests` | Test classes for different modules |
| `src/test/resources` | Test data and config files |
| `pom.xml` | Maven dependencies and build configuration |
| `testng.xml` | Test suite runner |
| `reports/` | HTML/Extent/Allure reports generated after execution |
